Game of Life

(Made with AngularJS 1.7)

This is a zero player game with the following rules:
- Any live cell (colored) with less than 2 live neighbours dies
- Any live cell (colored) with 2 or 3 live neighbours continues to be live
- Any live cell (colored) with more than 3 live neighbours dies (un-colored)
- Any dead cell (un-colored) with three live neighbours becomes a live cell
